Habilidades

HTML y CSS

Las dos tecnologías esenciales de la web moderna. Html define todo el contenido que vemos en las páginas, y CSS, mejora su diseño y apariencia.


JAVASCRIPT

Javascript es un lenguaje de programación usado inicialmente por navegadores web para crear experiencias de usuario dinámicas e interactivas. Hoy es un lenguaje que permite hacer cosas muy complejas en una página web. JavaScript es un lenguaje que permite crear el contenido dinámico que hemos referido anteriormente, además de controlar archivos, o imágenes. Pero no se queda solamente en estas características.


ANGULAR

Angular es un marco que simplifica el desarrollo de la aplicación y proporciona una gran flexibilidad de escritura de código. Desarrollado por Google, es uno de los frameworks Javascript más fuertes en la actualidad.


En todos mis proyectos, utilizo conjuntamente estas tecnologias, las cuales se adecuan a cada necesidad del cliente.
